LilyGO ESPHome YAML Helper

Paste the YAML generated by ESPHome Builder. This helper adds the Proflame2 LilyGO support while preserving your local Wi-Fi, API, OTA, and device identity settings.

1. Create device Create a new ESP32 device in ESPHome Builder so it generates your local secrets and starter YAML.
2. Paste YAML Paste the full generated YAML into the input box below.
3. Add support Click the button. The helper validates, patches, and re-validates the YAML.
4. Build in ESPHome Replace the ESPHome Builder YAML with the output and build/flash from ESPHome Builder.
